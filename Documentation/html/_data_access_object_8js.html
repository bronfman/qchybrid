Detailed Description

The DataAccessObject.js file contains an easy to use wrapper for the SQLite functionality of the device. The DataAccessObject constructor is used in the databaseDefinitions.js file.

function QueryResult ( )

Constructor for an object that contains the outcome of a database or web server data request. Depending on the type of database or web server request some of these values will or will not have values This object has the following attributes:

  • results An Array of Arrays that is a temporary 'table' of a query
  • errorMessage A String that contains an error message from the underlying database if there is one or 'not an error' if the SQL executed without error.
  • numRowsFetched An int indicating the number of rows in the results Array
  • fieldNames An Array of strings that are the field names that match the corresponding columns in the results temporary 'table'.
  • rowsAffected A int representing the records changed as the result of a non-select type statement.
    Returns:
    A QueryResult that contains all the results of the request.
