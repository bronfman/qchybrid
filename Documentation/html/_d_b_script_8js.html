Detailed Description

The DBScript.js file contains an easy to use object designed to accumulate SQL satements and parameters and then run them as a single method call. This is similar in behavior to SQL script files.

DBScript.executeSetDataScript()
The executeSetDataScript function runs all of the statements previously added to the script. The execution of these statements is done inside a transaction. If any one statement fails the entire set of changes is rolled back. If they all succeed the transaction is committed.

Parameters:
SQLA String that is a standard SQLite statement or has prepared statement form.
preparedStatementParametersAn optional Array of items to be matched with the '?' characters in the SQL string if the SQL has prepared statemetn form. The number of elements of this array must be the same as the number of '?' characters in the SQL string.
Returns:
nothing
