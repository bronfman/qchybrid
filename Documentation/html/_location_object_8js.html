LocationObject.getLocation(maxAgeSeconds, searchTimeoutSeconds)
The getLocation function is an asynchronous call designed to be called from within a stack.

Parameters:
maxAgeSecondsAn optional double that is the oldest, in seconds, that an old value for geolocation should be acceptably used. The default value is 0.1 seconds if no value or null is passed.
searchTimeoutSecondsAn optional double that is the maximum number of seconds that the geolocation code should search for the current location before terminating the search. The default value is 5 seconds if no value or null is passed.
Returns:
nothing

LocationObject trackLocation ( updateCommand  )

LocationObject.trackLocation(updateCommand)
The trackLocation function initiates a geo tracker that fires each time the device is moved in geologic space.

Parameters:
updateCommandA String that is a stack command. This stack will handle the notification of the geolocation change. The location will be found in the results attribute of a QueryResult that is passed to the stack as a parameter.
Returns:
nothing
